# 스터디하이

스터디하이는 아이에게 맞는 선생님을 연결하는 프리미엄 1:1 교육 브랜드 홈페이지입니다.

## 기술 스택

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui 스타일 컴포넌트
- Framer Motion
- next/image
- next/font

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.

## 검증

```bash
npm run lint
npm run build
```

## 배포 전 확인

- 실제 운영 도메인으로 `metadataBase`, `sitemap`, `robots` URL 변경
- 실제 카카오 채널 URL 변경
- 실제 상담 전화번호와 이메일 변경
